# Sistema de Lista de Presença (Check-in de Evento)

Este projeto é um sistema simples em Java para gerenciar a lista de presença de um evento via terminal.
Ele permite cadastrar participantes, realizar check-in e consultar informações de forma prática.

⚙️ Funcionalidades

✅ Adicionar participantes
📄 Listar participantes cadastrados
✔️ Realizar check-in (marcar presença)
❌ Remover participantes
🔍 Pesquisar participante pelo nome
🚪 Encerrar o sistema

Tecnologias utilizadas
Java

Projeto desenvolvido para prática de lógica de programação em Java.
